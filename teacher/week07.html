Meeting Time

The reading for this week introduces CSS transitions and animations. An example I often use is to start with these boxes and will change them (transform, size shape color, location) with transitions to demo the basics. I'll finish with a bouncing ball or make a box go around the content div (slide corner to corner all the way around). Then you might build something like this: PokeApi Demo to review Ajax and add the sliding transition effect.
