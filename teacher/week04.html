Instructor Preparation

Overview

The readings for this week cover a fairly wide variety of topics. Debugging is one, then Single Page Applications, and finally the HTML Template element. Plan on spending some time addressing questions.

The team activity continues our work at making the site dynamic by addressing the product list on the home page. It does introduce using a Template element. So you will want to include that in your demo this week.

Encourage the students also to record their questions while reading and give them to you to be addressed.

Meeting Time

To help the students prepare for the team activity you might do something like this SW Film information app. It uses the HTML template element to model one way of providing re-usable HTML to our code.

  1. Start by creating a new directory and files: index.html, main.js, SwapiHelper.js.
  2. Continue by brainstorming a list of tasks that need to be done. It might look something like this: 
    get list of films
output list of films
when a film is clicked:
    get detailed info on film
    output film details
    for each planet
        get planet info
        render planet with planet template
    for each species
        get species info
        render with template
    for each starship
        get info
        render with template
    for each vehicle
        get info
        render with template
  3. You might turn the students loose on the first two steps as review of what we have learned in previous weeks. They can use https://swapi.dev/api/films to get the initial info they need.
  4. Once they are done have a conversation about the method they wrote to get the info from the API. What happens if something goes wrong? Harden the method with try/catch. Also make sure that what the students came up with is re-usable...we will be making lots of different requests with that method.
  5. Add the code to handle a click on a film. Talk about how we will be using the same HTML to output all of the planets for example. In the past we have used functions to generate the HTML, we could also use a template in the HTML.
  6. Add a template element to the head of our index.html file.
