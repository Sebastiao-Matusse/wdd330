L08 End-of-Week

The team activity for this lesson is a continuation of the last activity. The students should have successfully submitted an order to the server, but we did not handle the response, or deal with validation to make sure the user gives us good information.

The reading for this lesson introduces JSON Web Tokens (JWT). The individual activity in the next lesson will address authentication.

Remind the students that their project proposals are due this week.

Encourage the students to record their questions while reading and give them to you to be addressed.

L07 Midweek

The reading for this lesson introduces CSS transitions and animations. As an example, I often use these boxes to start and will change them (transform, size, shape, color, location) with transitions to demo the basics. I'll finish with a bouncing ball or make a box go around the content div (slide corner to corner all the way around). Then you might build something like this: PokeApi Demo with animation to review Ajax and add the sliding transition effect.

Start by coming up with the list of steps we might follow to solve the problem. For example: 

  /*
  create an element with a list to hold the names of the Pokemon
  create a container to hold the details of the Pokemon
  place the containers side by side, make them each the size of the screen
  hide the details element (overflow:hidden on main)

  get the list of Pokemon
  display the list of Pokemon in the list element
      make sure to include the URL for the Pokemon details in the HTML
  attach a listener to the Pokemon list when clicked:
      get the details for the correct Pokemon
      insert the details into the details element
      slide the list and the details elements left 100vw
  Add a back button to the details element.
  attach a listener to the button, when clicked:
      slide list and details 100vw right
*/
